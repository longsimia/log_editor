## 編輯器功能

本工具為整理 TRPG 團錄的單檔網頁工具。具備以下功能：
- 修改團錄標題與訊息內容
- 插入、刪除、複製、拖曳排序訊息
- 插入分隔線與圖片
- 管理角色名字、顏色、頭貼與頭貼差分
- 管理頻道名稱、顏色、排序與群組顯示
- 建立目錄
- 預覽並匯出閱覽版 HTML

您可以將由 ccfolia 匯出的 HTML 團錄，直接於本工具中進行編輯、整理與排版，最後再匯出為適合閱讀與分享的 HTML 閱覽版。<br>
<br>
<br>

# 開始使用
1. 直接於 [github pages](https://longsimia.github.io/log_editor) 上使用，或下載 ```index.html``` 後於本機瀏覽器開啟。<br>
2. 將 ccfolia 匯出的 HTML 檔拖進頁面，或點選上傳區塊選擇檔案。<br>
   註：若使用的檔案為全頻道 [all] 記錄，則本工具會自動忽略來自 [other/閒聊/雑談] 頻道的發言。<br>
3. 檔案讀取完成後，工具會自動進入編輯畫面。<br>
<br>

## 建議使用流程
第一次整理團錄時，建議依照以下順序操作：
1. 上傳 ccfolia 團錄
2. 修改標題
3. 點擊 ```管理``` → ```角色管理```<br>
   整理角色名字、顏色與頭貼
4. 點擊 ```管理``` → ```頻道管理```<br>
   檢查頻道名稱與群組顯示
5. 刪修訊息
6. 建立目錄
7. 點擊 ```輸出預覽``` 檢查效果
8. 匯出閱覽版 HTML
<br>
<br>

## 功能說明

### 最上方工具列 (topbar)
- ```標題```：默認使用上傳的檔名。點擊可直接修改標題
- ```深色/淺色```：切換畫面主題色調
- ```目錄```：開啟並釘選右側目錄面板
- ```顯示```：可切換頭貼、角色名字與頻道的顯示
- ```管理```：可開啟```角色管理```與```頻道管理```
- ```輸出預覽```：可先行預覽最終匯出的閱覽版效果
<br>

#### 目錄
支援功能包括：
- 使用 ```自動偵測``` 建立目錄項目<br>
  註：偵測條件為1.發言角色為KP類角色 2.內文包括時間類文字
- 使用 ```＋ 新增``` 手動加入目錄<br>
  註：先點擊要加入目錄的訊息內文，再點擊 ```＋ 新增``` 即可將該則訊息加入目錄
- 直接修改目錄名稱
- 拖曳調整目錄順序
- 刪除單一項目
- 清除全部目錄
<br>

#### 角色管理
支援功能包括：
- 新增角色
- 修改角色名字與顏色
- 設定頭貼
- 設定隱藏名字時的角色內文顏色
<br>

#### 頻道管理
支援功能包括：
- 修改頻道名稱與標籤顏色
- 調整頻道排序
- 撰寫頻道說明
- 設定是否以群組樣式顯示
<br>
<br>

### 訊息操作工具列
訊息可以直接拖曳，以調整顯示順序。<br>
將滑鼠移到訊息上方時，右上角會出現以下操作按鈕：
- ```↑+```：在此訊息上方插入內容
- ```↓+```：在此訊息下方插入內容
- ```⧉```：複製這則訊息
- ```×```：刪除這則訊息
<br>
<br>

### 訊息文字格式工具列
選取訊息中的文字後，會出現浮動工具列，可使用以下格式功能：
- 粗體 ```Ctrl+B```
- 斜體 ```Ctrl+I```
- 刪除線 ```Ctrl+Shift+S```
- 文字顏色
- 插入超連結
- 置中
- 放大 / 縮小字號
<br>
<br>

## 細項功能

#### 單則訊息頭貼
設定角色頭貼差分後，可針對單則訊息，點擊該訊息頭貼選擇想套用的差分。<br>
<br>

#### 插入對話
插入時可設定該訊息所屬的頻道及角色。<br>
角色名字可以手動輸入，也可以直接從既有角色清單中選擇。<br>
<br>

#### 插入圖片
插入後可設定圖片尺寸、標記為爆雷圖片、放大檢視。<br>
<br>

#### 匯出閱覽版
匯出的閱覽版仍保留以下互動功能：
- 深色 / 淺色主題切換
- 圖片放大
- 爆雷圖片點擊顯示
- 目錄跳轉

另外，閱覽版 HTML 會盡可能保留原先設定的角色名字、頭貼、頻道內容、目錄等編輯資訊，以便再次匯入至本工具繼續編修。<br>
但第一次匯出時未顯示的頻道，重新匯入後將不會再出現於頻道清單中。<br>
<br>
<br>
<br>
<br>

# 注意事項
- 編輯內容都在瀏覽器中處理，重新整理頁面前請先匯出，以免內容遺失。
- 目前主要支援 ccfolia 匯出的 HTML 格式。
- 匯出格式目前為 HTML 閱覽版。
- 若來源 HTML 結構差異過大，可能無法正確解析。
